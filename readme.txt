# EffekseerForUE

This is a wrapper to play effects in UnrealEngine

At first, please read Help.

- Plugins
This directory contains plugin files to use EffekseerForUE

- Help
Help written about how to use on the online.

- EffekseerForUE.uproject
This uproject contains samples ofEffekseerForUE

■　EffekseerForUE

UnrealEngineでエフェクトを使用するためのラッパーです。

まずは、ヘルプを読みましょう。

・Plugins
EffekseerForUEを使用する時に使うプラグインが保存されています。

・Help
使い方について記述されているオンラインのヘルプです。

・EffekseerForUE.uproject
EffekseerForUEのサンプルが保存されています。
